(1) Press Win+R and type: regedit  

(2) In the Registry Editor left pane, expand the following directory: \HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsUpdate\UX\Settings  

(3) In the right pane, create a new DWORD (32-bit) Value  

(4) Rename it to: FlightSettingsMaxPauseDays  

(5) Double-click to open, change the Base to Decimal, and set the Value data to the number of days you want to pause updates, for example 20000, then click OK.  

(6) Finally, open the Start menu, go to Settings, open Windows Update, and set Pause updates to the period you just configured (if it does not appear, close Settings and open it again)  


Quick method  

(1) Search for CMD, find “Command Prompt,” right-click and select “Run as administrator”  

(2) Press Enter after typing: reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsUpdate\UX\Settings" /v FlightSettingsMaxPauseDays /t reg_dword /d 20000 /f  

(3) Finally, open the Start menu, go to Settings, open Windows Update, and set Pause updates to the period you just configured (if it does not appear, close Settings and open it again)
